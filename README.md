## Operadores aritméticos
```
 a = 5+2   - Soma
 b = 5-2   - Subtração
 c = 5*2   - Multiplicação
 d = 5/2   - Divisão
 e = 5**2  - Potência 
 f = 5//2  - Divisão inteira
 g = 5%2   - Resto da divisão
```
